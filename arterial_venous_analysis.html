Perioperative Plasma Inflammatory Markers: Arterial vs. Venous

Author

Gregory A. Chinn

Published

June 2, 2026

Study Overview

PI: Arun Prakash, UCSF Anesthesia & Perioperative Care
Assay run: December 2022 (65-plex cytokines); February 2022 (SomaScan)
Central question: Do plasma inflammatory marker levels differ between simultaneously collected arterial and venous blood during the perioperative period?

Physiologic rationale

The arterial sample (radial/femoral arterial line) is post-pulmonary — blood that has traversed the lung and entered the systemic circulation. The venous sample (central/peripheral IV) is pre-pulmonary — blood returning from the peripheral tissue bed. This paired design lets you measure the arteriovenous (A-V) gradient across both the lung and the systemic tissue beds:

  • Arterial > venous: the peripheral surgical/tissue bed is consuming the mediator
  • Venous > arterial: the peripheral tissue bed is releasing the mediator into venous return

Cohort

12 patients undergoing elective surgery under general anesthesia. One simultaneous arterial + venous draw per patient at a single post-induction / intraoperative time point. No pre-operative baseline was included in this submission.

# Standardize Y/N values: treat "y","Y" → TRUE; "N","N*" → FALSE; anything else → NA
clean_yn <- function(x) {
  x <- str_trim(str_to_upper(x))
  case_when(
    x == "Y"  ~ TRUE,
    x == "N"  ~ FALSE,
    TRUE       ~ NA   # covers NA, "N*", "UNKNOWN", "UNKNOWN (LIKELY Y)", "NA"
  )
}

Part 2: Arterial vs. Venous — Primary Analysis

Question: Is there a systematic difference in inflammatory marker concentrations between simultaneously collected arterial and venous blood?

Method: Paired Wilcoxon signed-rank test for each cytokine (arterial vs. venous, blocked on patient). Benjamin-Hochberg FDR correction across all 65 analytes.

Code 
# Separate arterial and venous, sorted identically by patient
art <- cytokine_avg |> filter(draw == "Arterial") |> arrange(patient)
ven <- cytokine_avg |> filter(draw == "Venous")   |> arrange(patient)

stopifnot(all(art$patient == ven$patient))

av_results <- map_dfr(cytokine_cols, function(cyt) {
  a <- art[[cyt]]
  v <- ven[[cyt]]

  test <- tryCatch(
    wilcox.test(a, v, paired = TRUE, exact = FALSE),
    error = function(e) list(statistic = NA, p.value = NA)
  )

  tibble(
    cytokine        = cyt,
    mean_arterial   = mean(a, na.rm = TRUE),
    mean_venous     = mean(v, na.rm = TRUE),
    median_arterial = median(a, na.rm = TRUE),
    median_venous   = median(v, na.rm = TRUE),
    log2FC          = log2((mean(a, na.rm = TRUE) + 0.1) /
                            (mean(v, na.rm = TRUE) + 0.1)),
    p_value         = test$p.value
  )
}) |>
  mutate(
    p_adj_BH           = p.adjust(p_value, method = "BH"),
    direction          = if_else(log2FC > 0, "Arterial > Venous", "Venous > Arterial"),
    significant_nominal = p_value < 0.05,
    significant_FDR    = p_adj_BH < 0.05
  ) |>
  arrange(p_value)

Part 3: Covariate Analysis — Cytokine Levels by Patient Characteristics

Question: Do baseline patient characteristics (obesity, hypothermia, hyperglycemia, sex, surgery type) associate with cytokine concentrations at the intraoperative time point?

Method: Venous samples used as the representative per-patient measurement (less affected by arterial catheter placement). Wilcoxon rank-sum test for binary covariates; Kruskal-Wallis for surgery type (3 groups). No FDR correction within this section — treat all results as exploratory given N = 12.

Helper: run all covariate tests

Code 
# Patient-level dataframe using venous concentrations
ven_pt <- cytokine_avg |>
  filter(draw == "Venous") |>
  select(patient, surgery_group, art_iv_same, white_race,
         male, obese, hypothermia, hyperglycemia,
         all_of(cytokine_cols))

# Binary covariate tests (Wilcoxon rank-sum)
binary_covariates <- c("obese", "hypothermia", "hyperglycemia",
                        "male", "white_race", "art_iv_same")

run_binary_tests <- function(df, covars, analytes) {
  map_dfr(covars, function(cov) {
    map_dfr(analytes, function(cyt) {
      group_vals <- df[[cov]]
      conc_vals  <- df[[cyt]]

      # Drop rows where covariate is NA
      keep <- !is.na(group_vals)
      g    <- group_vals[keep]
      y    <- conc_vals[keep]

      n_pos <- sum(g == TRUE,  na.rm = TRUE)
      n_neg <- sum(g == FALSE, na.rm = TRUE)

      if (n_pos < 2 || n_neg < 2) {
        return(tibble(covariate = cov, cytokine = cyt, p_value = NA,
                      median_pos = NA, median_neg = NA,
                      log2FC_pos_neg = NA, n_pos = n_pos, n_neg = n_neg))
      }

      test <- tryCatch(
        wilcox.test(y[g], y[!g], exact = FALSE),
        error = function(e) list(p.value = NA)
      )

      tibble(
        covariate      = cov,
        cytokine       = cyt,
        p_value        = test$p.value,
        median_pos     = median(y[g],  na.rm = TRUE),
        median_neg     = median(y[!g], na.rm = TRUE),
        log2FC_pos_neg = log2((median(y[g],  na.rm = TRUE) + 0.1) /
                               (median(y[!g], na.rm = TRUE) + 0.1)),
        n_pos          = n_pos,
        n_neg          = n_neg
      )
    })
  })
}

binary_results <- run_binary_tests(ven_pt, binary_covariates, cytokine_cols)

# Surgery type: Kruskal-Wallis (3 groups)
surgery_results <- map_dfr(cytokine_cols, function(cyt) {
  kw <- kruskal.test(ven_pt[[cyt]] ~ ven_pt$surgery_group)
  tibble(
    covariate = "surgery_group",
    cytokine  = cyt,
    p_value   = kw$p.value,
    n_pos     = NA_real_,
    n_neg     = NA_real_,
    log2FC_pos_neg = NA_real_
  )
})

all_covariate_results <- bind_rows(binary_results, surgery_results) |>
  mutate(p_adj_BH = p.adjust(p_value, method = "BH"), .by = covariate) |>

# Patient-level expression matrix using venous samples
soma_ven <- soma_samples |>
  filter(draw == "Venous") |>

expr_ven <- soma_ven |>
  select(all_of(analyte_cols)) |>
  as.matrix() |>
  log2()

# For each binary covariate, paired Wilcoxon or limma linear model
# With N = 12 venous samples, use simple linear model with limma for stability
soma_covariate_results <- map_dfr(binary_covariates, function(cov) {
  group <- soma_ven[[cov]]
  keep  <- !is.na(group)

  if (sum(keep) < 6 || length(unique(group[keep])) < 2) return(NULL)

  design_cov <- model.matrix(~ group[keep])
  fit_cov    <- lmFit(t(expr_ven[keep, ]), design_cov)
  fit_cov    <- eBayes(fit_cov)

  topTable(fit_cov, coef = 2, number = Inf, sort.by = "P") |>
    rownames_to_column("aptamer") |>
    mutate(covariate = cov) |>
    left_join(analyte_anno, by = c("aptamer" = "AptName")) |>
    as_tibble()
})

cat("SomaScan covariate hits (adj.P.Val < 0.05) per covariate:\n")
soma_covariate_results |>
  filter(adj.P.Val < 0.05) |>
  count(covariate) |>
  print()

Interpretation Notes

A-V gradient (dominant pattern): A consistent directionality across nearly all analytes suggests the peripheral tissue bed (surgical site + anesthetized tissues) is consuming inflammatory mediators from the arterial circulation during the intraoperative period. Power is insufficient at N = 12 to survive BH correction across 65 analytes.

IP-10/CXCL10: Consistently shows venous > arterial, indicating active release from the peripheral surgical bed. This is consistent with known IP-10 biology in perioperative inflammation (IFN-γ–inducible, elevated at sites of tissue injury).

Covariate analysis caveats: Binary subgroups are small (obesity: 4 yes / 8 no; hypothermia: ~7 yes / 5 no; hyperglycemia: 3 yes / 7 no; males: 4). All covariate findings should be treated as hypothesis-generating only. A future study with N ≥ 30 would be needed for adequate power.
