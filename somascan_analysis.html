Overview

SomaScan v4.1 PharmaServices run (PLT6574, 2022-02-26) on the same 24 EDTA plasma samples as the Luminex 65-plex study: 12 patients undergoing elective surgery, one simultaneous arterial (-4) and venous (-2) draw per patient. 7,596 aptamers measured; 7,481 pass ColCheck QC.

Analyses:

  1. Sample QC and PCA
  2. Arterial vs. venous: paired limma, 7,481 proteins
  3. Gene-set enrichment (GSEA) on the A-V ranking
  4. Covariate analysis on venous samples (obesity, hypothermia, hyperglycemia, sex)
  5. A-V gradient by covariate
  6. Cross-check against Luminex cytokine hits

ANOVA: effect of draw and surgery type on PC scores

Does draw type (arterial vs. venous) or surgery type explain variance in the top PCs? Each PC is modelled as a function of draw and surgery group. Because two samples come from each patient (arterial + venous), patient is included as a random effect via a mixed model (lme4::lmer). Results are compared with ANOVA (car::Anova, Type III) to correctly handle the unbalanced design.

Code 
library(lme4)
library(car)

pca_df <- pca_df |>
  mutate(
    draw          = factor(draw, levels = c("Venous", "Arterial")),
    surgery_group = factor(surgery_group)
  )

fit_pc <- function(pc_col) {
  form <- as.formula(paste0(pc_col, " ~ draw + surgery_group + (1 | patient)"))
  m    <- lmer(form, data = pca_df, REML = FALSE)
  anov <- car::Anova(m, type = "III")
  tibble(
    PC         = pc_col,
    term       = rownames(anov),
    Chisq      = round(anov$Chisq, 2),
    df         = anov$Df,
    p_value    = signif(anov$`Pr(>Chisq)`, 3)
  ) |>
    dplyr::filter(term != "(Intercept)")
}

pcs_to_test <- c("PC1", "PC2", "PC3", "PC4")
anova_results <- map_dfr(pcs_to_test, fit_pc)

3. Arterial vs. Venous — Paired limma

Design: block on patient (12 levels), coefficient of interest = draw (Arterial vs. Venous). Equivalent to a paired t-test across 7,481 proteins simultaneously. BH-FDR correction.

run_soma_covariate <- function(expr_mat, sample_df, cov_name, analyte_info) {
  group  <- sample_df[[cov_name]]
  keep   <- !is.na(group)

  if (sum(keep) < 6 || length(unique(group[keep])) < 2) {
    message("Skipping ", cov_name, ": insufficient non-NA observations")
    return(NULL)
  }

  design_cov <- model.matrix(~ group[keep])
  fit_cov    <- lmFit(t(expr_mat[keep, ]), design_cov)
  fit_cov    <- eBayes(fit_cov, trend = TRUE, robust = TRUE)

  topTable(fit_cov, coef = 2, number = Inf, sort.by = "P") |>
    rownames_to_column("AptName") |>
    as_tibble() |>
    left_join(analyte_info |> select(AptName, Target, EntrezGeneSymbol, UniProt),
              by = "AptName") |>
    mutate(
      covariate = cov_name,
      n_pos     = sum(group[keep] == TRUE,  na.rm = TRUE),
      n_neg     = sum(group[keep] == FALSE, na.rm = TRUE)
    ) |>
    arrange(P.Value)
}

Interpretation Notes

Sample flags: Four samples failed SomaScan RowCheck QC (000_0193-2, 000_0193-4, 000_0245-4, 000_0268-4). These are included in the paired analysis because ANML normalization has already corrected for sample-level signal variation; excluding them would eliminate patient 0193 entirely and reduce paired power. Results for those specific patients should be interpreted with caution.

Power: N = 12 paired samples against 7,481 proteins. The paired design captures the within-patient A-V contrast efficiently, but FDR correction across this many proteins is very conservative at this sample size. Nominal p-value threshold (p < 0.05) is the interpretively relevant cutoff; treat any FDR-significant finding as a robust signal and nominal hits as candidate proteins.

GSEA interpretation: GSEA on the limma t-statistic ranking is more powerful than individual protein testing because it aggregates signal across gene sets. Pathways enriched in arterial blood would suggest systemic arterial supply of those mediators; pathways enriched in venous return suggest peripheral tissue release.

Cross-platform consistency: IP-10/CXCL10 direction (venous > arterial) is the strongest a priori hypothesis from the Luminex data. Consistent direction in SomaScan (despite different aptamer-based measurement) would strengthen that interpretation substantially.
